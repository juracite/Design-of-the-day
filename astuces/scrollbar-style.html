Je vais donc utiliser pour cela, 2 éléments :
::-webkit-scrollbar-track {}
::-webkit-scrollbar-thumb {}

 - Le premier est pour le fond du slider, la zone de mouvement du slider.
- Le deuxieme, c'est le slider, celui que l'on fait défiler en scroll.

Ces 2 pseudo-elements ont des paramètres, je vais en utiliser quelques uns :

::-webkit-scrollbar-track {
-webkit-box-shadow: inset 0 0 6px rgba(0, 255, 193, 0.4);
border-radius: 10px;
visibility: visible;
}

::-webkit-scrollbar-thumb {
border-radius: 10px;
-webkit-box-shadow: inset 0 0 6px rgba(0, 255, 194, 1);
position: relative;
-webkit-transition: 0.5s ease 0.0s;
transition: 0.5s ease 0.0s;
}
